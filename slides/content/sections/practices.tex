\subsection{Get started}

\begin{frame}
\frametitle{Building AI Agent Systems}
\begin{block}{Hands-on focus}
From a toy Transformer to a chat agent with tools and a runnable UI stack.
\end{block}

\begin{itemize}
\item \textbf{LLM build and pretrain} \textemdash{} craft a minimal Transformer in \texttt{torch}, wire a tiny dataset, and pretrain/inspect with \texttt{deepspeed}.
\item \textbf{MCP server \& client} \textemdash{} define resources and tools, ship prompts, and exercise them via a toy command-line client.
\item \textbf{Agent workflow} \textemdash{} assemble a chat agent that hits the MCP tools, with a lightweight frontend, backend, and workflow engine.
\end{itemize}
\end{frame}
